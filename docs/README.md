### Common tasks, at a glance

- **Just make it work:** open the app, press **▶ Start**. ([User guide](USER_GUIDE.md#starting-and-stopping))
- **Use my own sounds:** Appearance → Sound → **Sound files…**, drop `.wav`s in. ([Sound](SOUNDS.md#using-your-own-sounds))
- **Change my city:** edit `location` in `config.json`. ([Configuration](CONFIGURATION.md#location))
- **Start with my computer:** Appearance → Engine → **Run automatically at login**. ([User guide](USER_GUIDE.md#running-automatically-at-login))
