### Common tasks, at a glance

- **Just make it work:** open the app; settings apply automatically. ([User guide](USER_GUIDE.md#automatic-apply))
- **Use my own sounds:** Settings → Sound → **Sound files…**, drop `.wav`s in. ([Sound](SOUNDS.md#using-your-own-sounds))
- **Change my city:** Settings → Engine → **City**. ([Configuration](CONFIGURATION.md#location))
- **Start with my computer:** Settings → Engine → **Run automatically at login**. ([User guide](USER_GUIDE.md#running-automatically-at-login))
